### 2-4. Skills & Expertise 레이아웃 개선

**문제:** Markdown 테이블에서 스킬 항목이 긴 경우 텍스트가 잘리거나 줄바꿈으로 가독성 저하.

**해결:** 단일 테이블 → 2열 카드 레이아웃으로 변경

```html
<div class="skill-grid">
  <div class="skill-card">
    <h3>Antibody Screening</h3>
    <ul>...</ul>
  </div>
  ...
</div>
```

**결과:** 각 카테고리가 독립 카드로 표시되어 가독성 개선.
